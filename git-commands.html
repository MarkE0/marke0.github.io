Git Commands Summary

Command/Task Command with Example Parameters Notes
Installing Git For Windows
Initialize a new repository git init Creates a new Git repository
Clone a repository git clone <repository-url> Copies an existing repository
Check status git status Displays the state of the working directory and staging area
Add files to staging area git add <file> Stages the specified file
Commit changes git commit -m "commit message" Records changes to the repository
Push changes git push Uploads local repository content to a remote repository
Pull changes git pull Fetches and integrates changes from a remote repository
Create a new branch git branch <branch-name> Creates a new branch
Switch to a branch git checkout <branch-name> Switches to the specified branch
Merge a branch git merge <branch-name> Merges the specified branch into the current branch
